# 機能名：コンパ一覧
# 機能ID：LDS0030
# 機能概要
* コンパの一覧を表示する
* マイページ(LDS0040)へ遷移する
* コンパ作成/編集画面(LDS0050)へ遷移する
* コンパ詳細/削除(LDS0060)へ遷移する
* 金額計算(電卓)(LDS0070)へ遷移する
# INPUT・OUTPUT
* "-"
# 画面設計
|No|名称|属性|イベント|補足|
|---|---|---|---|---|
|1|作成|Button|コンパを作るフォーム(LDS0011)に遷移||
|2|コンパ情報リスト|List|タップしたらコンパの詳細画面へ遷移||
